Projects

I have a lot of half-finished crap on GitHub. This is a curated selection of the more finished, actually useful stuff.

Most of it is quite old at this point but still relevant.

  • please is a command line utility that makes it easy to integrate web APIs into your shell scripts.
  • maur is a minimal tool for searching and installing packages from the AUR.
  • PS1 is my custom bash prompt with some nice features. I've been using this daily every since I wrote it.
  • Logoids is an Asteroids clone where you provide the Asteroid images.
  • Falsoyd is a terrible game I wrote around 2000CE. tbh I don't know if it even compiles on modern systems.
  • git-get is another tool I still use every day. Replaces git clone but organises repositories by their origin under ~/code.
  • git-cube reminds you to commit before you exit a shell session.
  • shue is a command line tool for modifying and converting colour values for use with CSS etc.
  • git-aux allows you to use git to manage replicating your dotfiles between machines and that sort of things.
  • Babel is an easy-to-parse and easy-to-produce data exchange format for easing communication between processes written in different programming languages. There are definitely beter choices for this now but I do have some ideas for a future iteration of this.
